# SaleeM 3.4 — Auto-scale ومحور الأسعار الذكي

- يظهر تنبيه عند كل اختيار لصورة ويطلب تأكيد تفعيل Auto-scale قبل بدء التحليل.
- يطلب محرك الرؤية ثلاثة أرقام من الصورة: السعر الحالي، أعلى سعر ظاهر، وأدنى سعر ظاهر.
- يستخدم الرسم هذه الحدود مع شموع السوق والدخول والوقف والأهداف لضبط محور الأسعار تلقائيًا.
- تُمنح جهة الهدف الخضراء مساحة أكبر من جهة الوقف الحمراء.
- يضاف هامش إضافي دائم فوق أعلى مستوى وتحت أدنى مستوى.
- Order Block أصبح عنصرًا ثانويًا وأخف بصريًا؛ يظهر أسفل السعر في الصعود أو أعلى السعر في الهبوط فقط.
- تحسنت رسالة الخطأ عند تعذر قراءة السعر لتطلب تفعيل Auto-scale وإظهار المحور كاملًا.
- أضيفت اختبارات لاتجاه المساحة الخضراء، الهوامش، وموضع Order Block.
