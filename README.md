# SaleeM Gold Analyst v2.0

تطبيق متخصص في تحليل الذهب XAUUSD على فريم خمس دقائق.

## طريقة العمل
- يقرأ آخر ساعتين من الصورة: 24 شمعة M5 من أقصى يمين الشارت.
- يعيد رسم الشارت من الصفر بصورة عمودية واضحة بدقة 1080×1920.
- يرسم أقرب دعمين ومقاومتين مهمين.
- سماكة خط الدعم أو المقاومة تعكس قوة المستوى.
- الكتابة بجوار المستويات صغيرة ونحيفة وواضحة.
- يختار سيناريو واحدًا فقط، وهو الأعلى احتمالًا.
- يعرض دخولًا أو دخولًا محتملًا، وقفًا بنيويًا، وTP1 وTP2 وTP3.
- سهم أخضر للصعود وسهم أحمر للهبوط.
- يضع ملاحظات مختصرة أسفل الشارت.
- الذاكرة للقراءة فقط، والصور مؤقتة ولا تحفظ.

## التشغيل
```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

يلزم ضبط `OPENAI_API_KEY`. ويمكن تغيير النموذج من `OPENAI_MODEL`، والافتراضي `gpt-4.1`.
