# SaleeM Gold Analyst v0.9

تطبيق متخصص في تحليل الذهب XAUUSD على فريم 5 دقائق وإخراج صورة واحدة مرسومة فوق الشارت المرفوع.

## جديد v0.9
- يرسم صفقة واحدة فقط: الأعلى احتمالًا.
- يعرض نسبة النجاح التقديرية للصفقة المختارة.
- وقف الخسارة مبني على بنية الشارت والذاكرة، وليس مسافة ثابتة.
- سهم أحمر للهبوط وسهم أخضر للصعود.
- دعم واضح لنماذج M وW عند ظهورها.
- منطقة الهدف خضراء شفافة داخل الشارت.
- لا يوجد أرشيف للتحليلات أو النتائج.
