# SaleeM 3.3 — تصحيح المنطق والرسم

- إزالة الانحياز الافتراضي إلى BUY أو SELL، وتطبيع احتمالي الشراء والبيع معًا.
- إضافة حالة مراقبة حقيقية عند ضعف الإشارة أو تعارض H4 وH1.
- تشديد شروط «مؤكد» وربطها بتوافق الفريمات وصحة هندسة الصفقة.
- اختصار شروط الدخول وتثبيت اتجاه الوقف والأهداف.
- إصلاح النص العربي المختلط بالأرقام وTP باستخدام BiDi/RAQM.
- ترتيب جدول ملاحظات التحليل في عمودين ثابتين.
- استخدام البرتقالي للمقاومة والأزرق السماوي للدعم.
- منع تكرار أسعار المستويات على يمين الشارت.
- جعل خطوط TP متصلة ورفيعة وبطاقاتها أصغر.
- تكبير السهم ورفعه فوق مناطق الصفقة.
- فصل منطقتي الوقف والهدف عند خط الدخول دون تداخل لوني.
- إخفاء Order Block وFVG القديمة أو الضعيفة، وتمديد المناطق المفيدة نحو التحليل.
- مزامنة صناديق الجلسات مع وقت أحدث شمعة، مع إمكانية تعديلها من Railway.
- إزالة ساعة الجهاز والبطارية وتنظيف الملفات المؤقتة والقديمة.
